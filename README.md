## 路由美化插件


### 项目介绍

*  基于lake-admin的路由美化插件，自定义系统的url


### 使用方法 

*  composer安装
    * `composer require lake/lake-admin-addon-lroute`
*  本地安装
    * 后台 `本地模块->模块管理->本地安装` 上传模块或者手动将模块上传到 `/addon` 目录
    * 手动放置模块或者上传模块请确保目录为 `lroute/Service.php`
*  最后在 `本地模块->模块管理->全部` 安装添加的模块
